# IMDB-SwiftUI
Example project of [SwiftUI](https://developer.apple.com/xcode/swiftui) and [Combine](https://developer.apple.com/documentation/combine) using the IMDB-API.

Requirements
- Swift5.1 Beta
- Xcode11.0 Beta
- iOS 13.0 Beta
